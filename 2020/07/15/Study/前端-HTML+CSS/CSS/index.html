驼峰标识

  • 小驼峰：第一个单词的首字母小写，后面遇到一个单词首字母就大写
    • html class 小驼峰
    • js 函数名字
  • 大驼峰：所有单词的首字母都大写
    • js中定义类的名字
  • 设置网页图标
    • link元素
    • link元素的属性不能省略，用来指定文档于连接资源的关系
    • 一般rel若确定，相应的type也会默认确定，所以可以省略type
    • 网页图片支持的图片格式是ico、png，常用大小是1616、2424、36*36

2.最常用的CSS属性

  • color：前景色
  • font-size
    • 单位主要为px，还有em、rem、百分比
  • background-color
  • width、height
    • 对某些元素无用，如span等非替换行内元素
  • rgb颜色
    • red(0255)、green(0255)、blue(0~255)的结合
  • rgba颜色
    • 多了一个alpha(0~1)
      • 0：透明
      • 1：不透明

3.文本属性

  • text-decoration 用于设置文字的装饰线
    • none：无任何装饰线
      • 可用于去除a元素（超链接）默认的下划线
    • underline：下划线
    • overline：上划线
    • line-through：中划线
  • letter-spacing 用于设置字母间的间距 在style中设置
  • word-spacing 用于设置不同单词间的间距 在style中设置
  • text-transform 用于设置文字的大小写转化
    • capitalize：将每个单词的首字母变为大写
    • uppercase：将每个单词的所有字符变为大写
    • lowercase：将每个单词的所有字符变为小写
    • none：无影响
  • text-indent 用于设置首行缩进
    • 缩进2em正好是缩进两个文字
      • em继承父类字体大小，即X em代表几个父类字体大小的缩进
  • text-align可用于设置元素内容在元素中的水平对齐方式
    • left：左对齐
    • right：右对齐
    • center：居中显示
    • justify：左右两端对齐（对最后一行没有效果，如果想将最后一行也对齐，使用text-align-last： justify设置）

特殊的块级盒子 div：

如果是div包裹div，那么默认里面的div区块和外面的div区块是一个块，对内部区块的居中只随着外部div区块的设置而改动

解决方法就是使用display属性进行设置

4.字体

  • 字体大小（会继承）
    • px：像素大小 最常用
    • em，rem（用的比em多）
    • 百分比
  • 字体名称 font-family
    • 支持中英文字体名称
    • 一般中文字体适合英文和中文，英文字体只适合英文
    • 如果开发中中英文使用不同字体，建议英文字体在前，中文在后
    • 也可以使用网络字体（老师后续讲）
  • 文字粗细 font-weight
    • 100、200、…、900，每一个数字标识一个重量
    • normal：400，bold：700
  • 文字倾斜 font-stlye
    • italic、oblique 均表示斜体，前者必须字体支持斜体，后者就是简单的文本倾斜
    • 使用i元素也可以进行斜体
  • 影响小写字母的显示形式 font-variant
    • normal：常规显示
    • small-caps：将小写字母替换为缩小过的大写字母
    • 对比图：
  • 文本的最小行高 line-height
    • 行高理解为一行文字所占据的高度
    • 行高的严格定义：两行文字基线（baseline）之间的间距
    • 基线：与小写字母x最底部对齐的线
    • 示例：
  • height和line-height的区别
    • height：元素的整体高度
    • line-height：元素中每一行文字所占据的高度
    • 应用：假设div（高度比较高，即一大块那种）中只有一行文字，让这行文字在div内部垂直居中
  • 缩写属性 font
    • 可以将所有属性写到一个缩写属性里
    • 顺序为：font-style font-variant font-weight font-size/line-height font-family （斜体，字母显示形式，字体粗细，字体大小/行高，字体）
    • font-style variant weight可以随意调换顺序，也可以省略
    • /line-height可以省略，如果不省略，必须跟在font-size后面
    • font-size font-family不可以调换顺序，不可以省略
  • outline
    • 显示元素的外轮廓
    • 不占用空间
    • 默认显示在border的外面
    • 属性和border类似
