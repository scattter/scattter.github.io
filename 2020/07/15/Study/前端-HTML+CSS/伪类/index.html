一.伪类和伪元素

1.常见的伪类

  • 动态伪类：在编写程序时最好按照下面顺序
    • :link 未访问的链接
    • :visited 已访问的链接
    • :focus 指当前拥有输入焦点的元素（能接收键盘输入）
      • 取消聚焦的话可以使用outline&tabindex=-1
    • :hover 鼠标挪动到链接上
    • :active 激活的链接（鼠标在连接上长按未松开）
    • hover和active还可以运用到其他元素上
  • 目标伪类（不常见）:target
  • 语言伪类（不常见）:lang
  • 元素状态伪类 :enabled :disabled :checked
  • 结构伪类
    • :nth-child() :nth-last-child() :nth-of-type() :nth-last-of-type()
    • :first-child :last-child :first-of-type last-of-type
    • :root :only-chid :only-of-type :empty
  • 否定伪类
    • :not()

2.伪类详述

  • :disabled 不显示button的效果

  • :nth-child()

    • 表示对第几个字类进行操作
    • :nth-child(n)

  • :nth-last-child()

    • 表示对倒数第几个字类进行操作
    • nth-last-child(-n+2) 代表最后两个子元素

  • :nth-of-type()

    • 和:nth-child()有点类似，但是它只看同类型的第n个元素，如p:nth-of-type()只看子元素中的第n个p元素，而p:nth-child()是子元素中的第n个元素

  • :nth-last-of-type()

    • 和:nth-last-child()同理

  • :first-child 和 :last-child等同于:nth-child(1)与:nth-last-child(1) 第一个和最后一个

  • :first-of-type 和 :last-of-type等同于:nth-of-type(1)与:nth-last-of-type(1)

  • :only-chid 是父元素中唯一的子元素

  • :only-of-type 是父元素中唯一的这种类型的子元素

  • :root 根元素，就是html元素

  • :empty 空的元素

  • :not() 否定伪类，只支持单选择器，不支持组合选择器，如div.class

3.常见的伪元素

一般使用双冒号以区别伪类

  • ::first-line ::first-letter不经常用
  • ::before ::after经常用到

4.伪元素详述

伪元素是行内元素，其特性不能改，如width，如果想进行更改使用display: inline-block

  • ::first-line ::first-letter
    • 分别表示对第一行与段落的首个字母或文字进行修饰
    • first-line可应用的元素为：
      • 字体属性、颜色属性、背景属性
      • word-spacing、letter-spacing、text-decoration、text-transform、line-height
    • first-letter同理
  • ::before ::after
    • 分别在元素之前和之后插入文字、图片
    • content元素不能省略
