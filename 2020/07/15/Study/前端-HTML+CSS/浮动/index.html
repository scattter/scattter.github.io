一.浮动

CSS中常用的三种方法对元素进行定位\布局

  • normal flow:标准流/常规流/文档流
  • absolute positioning:绝对定位
  • float:浮动

绝对定位/浮动都会让元素脱离标准流,以达到灵活布局的效果

  • 浮动的常用值

    • none:不浮动,默认值
    • left:向左浮动
    • right:向右浮动

  • 浮动的规则

    • 元素浮动后

      • 脱离标准流
      • 朝着向左或向右方向移动,知道自己边界紧贴着包含块(一般是父元素)或者其他浮动元素的边界

    • 1.定位元素会层叠到浮动元素的上面

    • 2.浮动元素不能与行内级内容层叠,行内级内容将会被浮动元素推出

      • 比如行内级元素、 inline- block元素、块级元素的文字内容

      • 利用此特性,可以轻松实现图文环绕效果

    • 3.行内级元素、 inline- block元素浮动后,其顶部将与所在行的顶部对齐

    • 4.浮动元素不能超过其父元素的范围

    • 5.浮动元素直接不能层叠

    • 6.浮动元素的顶端不能超过包含块的顶端，也不能超过之前所有浮动元素的顶端

  • 浮动的应用场景

    • 解决行内级元素、inline-block元素的水平间隙问题
    • 布局

浮动应用中出现的问题:在进行布局时，同一行多个元素摆放后，设置margin-right让他们之间产生间距，那么总有一个元素设置的margin-right是多余的

解决办法：

  • 每一行最后一个元素，总是加一个class类，通过类选择器去除这个margin-right
  • 通过伪类选择器
    • 但是错在浏览器兼容性问题，IE8不兼容
  • 增加新的包含块
    • .container固定的宽度>.wrap+负margin-right>使得.wrap宽度变宽，能够包含更多的包含块

margin-right为负值时，还可以解决：两个浮动元素边框叠加的问题

二.transform

CSS transform属性允许你旋转,缩放,倾斜或平移给定元素

常见函数

  • 平移：translate(x,y)
    • 一个值时设置x轴的位移,两个时设置x和y的位移
    • 数字:100px
    • 百分比:参照元素本身
  • 缩放：scale
    • 缩放的是两个轴
    • 数字:1,保持不变 2,放大一倍 0.5,缩小一半
    • 不支持百分比
  • 旋转：rotate(deg)
    • 旋转,也与transfrm-origin有关
    • 倾斜：skew(deg,deg)
    • 一个值是x轴倾斜,两个则是x和y的倾斜
  • transfrm-origin
    • 用来设置变形的原点/起点
    • 一个值,设置x轴的远点;两个值,设置x和y的原点
    • 必须是<length>,<precentage>,或<left,center,right,top,bottom>关键字中的一个
      • <left,center,right,top,bottom>关键字
      • length:从左上角开始
      • 百分比:参考元素本身大小
