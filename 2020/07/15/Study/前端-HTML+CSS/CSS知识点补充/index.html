4.HTML5-CSS3

  • HTML5

    • 狭义的HTML5说的是HTML新的元素和属性
    • 广义的HTML5说的是HTML5新的标准,包括最新的HTML元素+CSS新特性+ Javascript

  • CSS3目前并不存在真正意义上的CSS3,只是对某些 Module Level3的统称

    • 关于某些 Level3是否已经称为标准,还是要多查阅文档
    • 另外可以通过 canuse查看浏览器的兼容性问题
  • HTML5 新增一些语义化的元素
  • 新增的一些元素
  • input元素属性的增加
    • autofocus:自动聚焦,即进入网页后自动聚焦到输入框
    • placeholder:输入框的占位文字
    • multiple:多选
    • input->type:data 可以自己选择时间

    • 动画时间
    • 动画样式(线性等)
