input元素

一.表单元素

  • input的常用属性

    • type: input的类型
      • text:文本输入框(明文输入)
      • password:文本输入框(密文输入)
      • radio:单选框
      • checkbox:复选框
      • button:按钮
      • reset:重置
      • submit:提交表单数据给服务器
      • file:文件上传
    • maxlength:允许输入的最大字数(比如手机号只能输入11位)
    • placeholder:占位文字(输入框内的字,如:请输入手机号)
    • readonly:只读文字(输入框内不能输入文字)
    • disabled:禁用(按钮禁用,如发送验证码按钮,点击一次后不能立马使用)
    • checked:默认被选中,只有当type为radio或checkbox时可用
    • autofocus:当页面加载时,自动聚焦
    • name:名字,用于区分数据类型
    • value:取值
    • form:设置所属的form元素,填写form元素的id
    • 去除input的Tab键选中效果
      • style中input {outline:none}
      • 或者是将tabindex属性值设置为-1

  • 布尔属性

    • 布尔属性可以没有属性值,写上属性名就代表使用这个属性
    • 常见的布尔属性有 disabled、 checked、 readonly、 multiple, autofocus、 selected
      • multiple:显示多个选项
      • size:显示多少项
      • selected:默认选中该项

  • textarea属性

    • cols:列数
    • rows:行数
    • resize:缩放的CSS设置
      • 禁止缩放,resize:none
      • 水平缩放,resize:horizontal
      • 垂直缩放,resize:vertical

  • fieldset和legend

    • 设置外边框

  • 表单提交

    • ​

  • form属性

    • action:用于提交表单数据的请求URL
    • method:get和post的提交请求方式
    • target:在什么地方打开URL(参考a元素的target,选择是当前也打开或者是打开新的标签页)
    • accept-charset:规定表单提交时使用的字符编码(默认值UNKNOWN,和文档相同的编码)
    • enctype:规定了在向服务器发送表单数据之前如何对数据进行编码
      • multipart/form-data:文件上传时必须为这个值,并且method必须是post
      • text/plain:普通文本传输
      • 还有一个默认的,不用记

二.元素类型

  • 根据元素能否在同一行显示,可分为两类
    • 块级元素
      • 独占父元素一行
    • 行内级元素
      • 多个行内级元素可以在父元素的同一行中显示
  • 根据元素内容是否可被替换,分为两类
    • 替换元素
      • 元素本身没有实际内容,浏览器根据元素的类型和属性,来决定元素的具体显示内容,如img,iframe,input
    • 非替元素
      • 和替换元素相反,元素本身是有实际内容的,浏览器会直接将其内容显示出来,而不需要根据元素类型和属性来判断到底显示什么内容

行内级替换元素可以随意设置宽高

块级元素的display属性为block

  • CSS属性-display

    • CSS中有个 display属性,能修改元素的显示类型,有4个常用值
      • block:让元素显示为块级元素
      • inline:让元素显示为行内级元素(默认属性)
      • none:隐藏元素
      • inline-block:让元素同时具备行内级元素和块级元素的特征

  • visibility

    • 控制元素的可见性
      • 默认为visible:显示元素
      • hidden:隐藏元素 和display的隐藏相比,仅仅是看不见了,但是框还在

  • overflow

    • 用于控制内容溢出时的行为
      • visible:溢出的内容照样可见
      • hidden:溢出的内容直接裁剪
      • scroll:溢岀的内容被裁剪,但可以通过滚动机制查看
        • 会一直显示滚动条区域,滚动条区域占用的空间属于wdth、 height
      • auto:自动根据内容是否溢出来决定是否提供滚动机制
    • 还有 overflow-x、 overflow-y两个属性,可以分别设置水平垂直方向
      (建议还是直接使用 overflow,因为目前 overflow-x、 overflow-y还没有成为标准,浏览器可能不支持)

  • float浮动

    • 可以用来去除空格
    • style中设置
