1.全局变量和局部变量

  • 全局变量和局部变量定义:

    • 全局变量是在全局作用域下的变量,在全局下都可以使用
      • script标签的第一层
      • 特殊情况是:在函数内部没有var声明直接赋值,也属于全局变量
    • 局部变量是在局部作用域下的变量,仅能在函数内部使用

  • 从执行效率看全局变量和局部变量

    • 全局变量只有浏览器关闭的时候才会销毁,占用内存资源
    • 局部变量在程序执行完毕后就会销毁,节约内存资源

  • 作用域链

    • 内部函数访问外部函数的变量,采取的是链式查找的方式来决定取那个值这种结构我们称为作用域链
    • 依次向外一层查找变量

  • 预解析和代码执行

    1. 预解析js引擎会把js里面所有的var(变量提升)还有 function(函数提升)提升到当前作用域的最前面
      1. 变量提升就是把所有的变量声明提升到当前的作用域最前面不提升赋值操作

      3. 函数提升就是把所有的函数声明提升到当前作用域的最前面不调用函数

3.内置对象

内置对象就是指JS语言自带的一些对象,这些对象供开发者使用,并提供了一些常用的或是最基本而必要的功能(属性和方法

  • Math对象

  • Date()对象

    • 该对象是一个构造函数,必须使用new来调用创建我们的日期对象
    • 即var date = new Date()

  • 数组对象

    • 检测是否为数组

      • 使用instanceof: arr instanceof Array
      • 使用Array.isArray(arr) 来进行判断 (这是H5新增的标准)

    • 数组添加元素

      • push() 在数组末尾添加元素
      • unshift() 在数组开头添加元素
      • 两者返回值都是新数组的长度

    • 数组删除元素

      • pop() 删除数组的末尾元素
      • shift() 删除数组的开头元素
      • 返回的是删除的元素 pop就是出栈

    • 数组的排序和反转

      • arr.sort(function(a,b) {
    return a-b; //升序排列
})

      • 反转使用arr.reverse()即可

  • 字符串对象

    • 对象和复杂数据类型才有属性和方法
      • str/Number/Boolean是特殊的.因为str.length js底层会把str包装为复杂数据类型
    • 字符串的不可变
      • 指是字符串里面的值不可变,虽然看上去可以改变内容,但事实是地址变了,内存中新开辟了一个内存空间
      • 因为字符串不可变,所以不要大量的拼接字符串
    • 字符串指定位置字符
      • 使用arr.charAt()
      • 使用charCodeAt() 返回相应索引号的字符ASCII值
      • 使用str[index] 这是H5新增的

  • 字符串的截取

    • concat用于连接两个或多个字符串 但是一般用’+’更多
    • substr (start,length) 从start初开始,截取长度为length的字符串
    • split分割数组为字符串

  • 简单数据类型和复杂数据类型

    • 简单数据类型:string number boolean undefined nul
      • null 返回的数据类型返回的是空的对象
      • 简单数据类型都放在栈里面
    • 复杂数据类型:使用new关键字创建的对象类型都是 也叫引用数据类型
      • 复杂数据类型都是存储在堆里面
      • 首先在栈里存放地址,然后地址指向堆里面的数据
